# MLBox

This is the MLBox Prototype.

## Installation

To install:

```sh
git clone https://github.com/xyhuang/mlbox.git
cd mlbox

pip install .
```

To uninstall:

```sh
pip uninstall mlbox
```

## Run an MLBox

```sh
mlbox run --config YOUR_CONFIG_FILE
```
